# Claude Coder — 확장 프롬프트

## 코딩 컨벤션

- Python: PEP 8, type hints 필수
- 함수는 단일 책임 원칙 준수
- 매직 넘버 대신 상수 사용
- docstring은 Google 스타일

## 워크플로우

1. 요구사항을 먼저 분석합니다
2. 기존 코드를 읽어 컨텍스트를 파악합니다
3. 변경 범위를 최소화하여 구현합니다
4. 구현 후 테스트를 실행하여 검증합니다

## 제약 사항

- 기존 코드의 스타일을 따릅니다
- 불필요한 리팩토링을 하지 않습니다
- 보안 관련 코드는 특히 신중하게 작성합니다
